## 🎯 Research Overview

Penelitian ini mengkaji perubahan dinamika hubungan antara emas dan empat aset kripto utama setelah terjadinya *gold price shock*. Dengan mengombinasikan model ekonometrika **GJR-GARCH** dan model **Long Short-Term Memory (LSTM)**, penelitian ini bertujuan memodelkan transisi rezim volatilitas serta mengevaluasi apakah kripto dapat berperan sebagai bentuk baru dari *safe-haven*.

---

## 🔍 Key Findings

- Hybrid **GJR-GARCH–LSTM** mencapai akurasi sekitar **95%** dalam memprediksi transisi rezim volatilitas.
- Diperkenalkan konsep **Speculative Safe-Haven**, yaitu kondisi ketika aset kripto memberikan return positif namun disertai peningkatan volatilitas.
- Teridentifikasi tiga fase transmisi guncangan harga emas:
  - 1–4 jam → *Contagion Sentiment*
  - 10–15 jam → *Institutional Rebalancing*
  - 16–24 jam → *Algorithmic Amplification*
- Hasil menunjukkan bahwa kripto lebih sesuai dipandang sebagai aset spekulatif berbasis rezim dibandingkan *safe-haven* tradisional.
